Level 1 (CST8215)-Introduction to Database

You will learn about Database Management System by practicing SQL queries, drawing Entity Relationship Diagrams. You will be creating databases using the concept of Normalization and be able to maintain and modify the databases.

Level 1 (CST8116)-Introduction to Computer Programming

This course is about understanding the basic fundamentals of programming. As a student, you get to develop basic knowledge of how to write programs, create pseudocode, flowcharts and practice Java Programming language. You will also learn about constructors, variables, one-dimensional array, inputs/outputs, abstraction, encapsulation and inheritance.

Level 1 (CST8300) Achieving Succes in Changing Environment

Teaches about changes in technology that is challenging us to find our place in the industry. This course teaches about applying critical thinking and decision-making tools to help solve some of the important issues in the society with its competing interests.

Level 1 (CST8101) Computer Essentials

This course teaches about the software, hardware and different components about laptop. It helps to develop technical programming skills as well as gain knowledge about basic materials and troubleshoot different problems that occur in computer system. You get to learn how to create backups, manage files, protect computer from viruses.

Level 1 (MAT8001C) Tech Math for Computer Science

This course is about algebraic and transcendental functions that are prerequisite to Calculus. Students learn to perform algebraic expressions, solve algebraic equations and linear systems and learn the properties of algebraic and transcendental functions. It will cover computer number systems in addition to Boolean algebra and logic to help solve problems involving computer systems. The course also teaches the addition and subtraction of vectors using vector components.
